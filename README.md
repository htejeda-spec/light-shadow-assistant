
# Interactive Light and Shadow Assistant

This project is an educational web-based application designed to help beginner artists understand how light interacts with three-dimensional forms.

## Features
- Sphere, cube, and cylinder selection
- Real-time lighting adjustment
- Lighting presets
- Visual light direction indicator
- Cast shadows

## Technologies Used
- HTML
- CSS
- JavaScript
- Three.js

## How to Run
1. Download the project files.
2. Open the project folder in Visual Studio Code.
3. Run the application using Live Server or open `index.html` in a browser.
